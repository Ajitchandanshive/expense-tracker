# Quick Start Guide

## 🚀 Get Started in 30 Seconds

### Windows

1. **Double-click** `run.bat` in the folder
2. Browser opens automatically at `http://localhost:8000`
3. Start adding expenses!

### Mac/Linux

1. **Open Terminal** and navigate to the folder
2. **Run**: `bash run.sh`
3. Browser opens at `http://localhost:8000`
4. Start adding expenses!

---

## 📱 Install on Mobile

### iPhone/iPad (iOS)

1. Open Safari browser
2. Enter: `http://[YOUR_COMPUTER_IP]:8000`
   - Find your IP: Open Terminal and run `ipconfig getifaddr en0` (Mac) or check router
3. Tap **Share** button (↗️)
4. Tap **Add to Home Screen**
5. Name it and tap **Add**
6. Icon appears on your home screen!

### Android Phone

1. Open Chrome browser
2. Enter: `http://[YOUR_COMPUTER_IP]:8000`
   - Find your IP: Go to Settings > About phone
3. Tap **Menu** (three dots) > **Install app**
4. Confirm installation
5. App appears in your apps!

---

## 💡 Basic Usage

### Adding an Expense

1. Fill in **Description** (what did you spend on?)
2. Enter **Amount** (in ₹)
3. Select **Category** (Food, Transport, etc.)
4. Pick **Date** (defaults to today)
5. Click **Add Expense**

### Viewing Your Spending

- **Monthly Total** - Total spent this month
- **Average Daily** - How much per day on average
- **Category Breakdown** - Spending per category (pie chart equivalent)

### Managing Expenses

- **Filter** - Select a category to see only those expenses
- **Delete** - Remove an expense by clicking the delete button
- **Export** - Download as CSV file for Excel/Sheets

## ❓ Common Questions

**Q: Do I need internet?**
A: No! Works completely offline. Your data is saved on your device.

**Q: Where is my data stored?**
A: Your device's local storage. Private, never sent anywhere.

**Q: Can I use on multiple devices?**
A: Yes! Each device has its own data. Export CSV to transfer between devices.

**Q: What if I clear my browser cache?**
A: Your expenses will be deleted. Export regularly as backup!

**Q: Will my data sync across devices?**
A: Not automatically. Export/Import CSV files to sync manually.

## 🎯 Tips & Tricks

1. **Use consistent descriptions** - "Starbucks", "Coffee Shop" are the same, be consistent
2. **Daily entry habit** - Add expenses same day, easier to remember
3. **Monthly review** - Check spending patterns at month end
4. **Export regularly** - Keep backups of important months
5. **Use categories** - Helps identify where your money goes

## ⚙️ Troubleshooting

**Problem: Browser won't open**
- Manual: Open browser and type `http://localhost:8000`

**Problem: Can't see from phone**
- Make sure computer and phone are on **same WiFi**
- Check firewall isn't blocking port 8000
- Try using computer's local IP (e.g., 192.168.1.100:8000)

**Problem: Data not saving**
- Clear browser cache
- Try private/incognito mode
- Try different browser (Chrome, Firefox, Safari)

**Problem: App is slow**
- Clear browser cache
- Close other tabs
- Restart browser
